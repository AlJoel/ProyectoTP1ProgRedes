# TP Cliente-Servidor

Trabajo práctico de programación sobre redes.

## Funcionalidad

- Servidor con sockets en localhost:5000
- Cliente que envía múltiples mensajes
- Almacenamiento en SQLite
- Respuesta con timestamp

## Cómo ejecutar

1. Ejecutar servidor:
   python server.py

2. Ejecutar cliente:
   python client.py; para finalizar escribir "éxito"

3. Opcional, si se quiere ver los mensajes ejecutar:
   python ver_mensajes.py
